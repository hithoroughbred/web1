HTML

Hypertext Markup Language< is the standard markup language for documents designed to be displayed in a web browse HTML paragraph HTML tag is used to define a paragraph in a webpage. It is a Paired Tag, i.e. it comes with an opening and a closing tag.

A tag is very important tag, as all the content written on a website needs to get formatted in the form of paragraphs.

Browsers automatically add blank lines above and below a paragraph to seaprate it from other content or othe paragraphs on the page. HTML Paragraphs are block level elements, i.e., a new paragraph will always start from a new line. Also, Paragraph tags gets automatically closed if another block-element gets parsed before the tag.
